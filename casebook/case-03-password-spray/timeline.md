# Timeline (example format)

> Use exact times if you have them. For portfolio cases, this is a realistic ticket timeline format.

- **T0** — Alert triggered in SIEM
- **T0 + 5m** — Tier‑1 validated entity context (user/host/IP)
- **T0 + 15m** — Enrichment completed (reputation/baseline)
- **T0 + 25m** — Decision made and escalation prepared
- **T0 + 30m** — Escalated to Tier‑2 with evidence package
